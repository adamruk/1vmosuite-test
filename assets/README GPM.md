# 🚀 GPM Profile Manager

## 🎯 User Guide

### Profile Controls
- 📁 Load GPM Profiles: Load list of GPM profiles
- Sort Options:
  - Sort by Name: Sort profiles alphabetically
  - Sort by ID: Sort profiles by ID
- ❓ Help: Show user guide

### Profile Management
- Profile List: Shows all available GPM profiles
  - Checkbox: Select profiles to manage
  - Profile Name: Display name of the profile
  - Profile ID: Unique identifier

### Action Controls
- Start Selected Profiles: Activate selected GPM profiles
- Post Content: Enter Facebook post content
- Post from Selected Profiles: Post content from selected profiles

### Status Display
- Log Window: Shows real-time operation logs
  - Profile activation status
  - Post operation status
  - Error messages if any

### Result List
Shows information about processed profiles:
- Profile: Profile name and ID
- Status:
  - 🟡 Processing: Operation in progress
  - 🟢 Completed: Operation successful
  - 🔴 Error: Operation failed (with error details)
- Time: Operation timestamp

### Tips
1. Always check profile status before posting
2. Wait for profile activation to complete
3. Monitor log window for operation status
4. Check result list for detailed information
